# **App Name**: Mp4movies

## Core Features:

- Movie Listing: Display a listing of all movies fetched from movies.json with SSR.
- Movie Detail Pages: Individual movie detail pages generated server-side, fetching data from movies.json using the slug.
- Theme Toggle: Allow users to toggle between light and dark themes, with dark mode as the default, using Context API or Zustand for state management.
- SEO Optimization: Implement SEO-optimized metadata including title, description, and canonical URLs for each page, along with Open Graph and Twitter card tags.
- Framer Motion Animations: Employ Framer Motion for page transitions, card hover effects, and staggered list animations.
- Static Pages: Generate the 'About', 'Contact', 'Privacy Policy', and 'Terms of Service' static pages, linking them from the navigation and/or footer of the app.

## Style Guidelines:

- Primary color: Orange (#FD5E1E), inspired by HiAnime's accent color, for a vibrant and energetic feel.
- Background color: Dark gray (#111315), close to HiAnime's dark theme, providing a modern and immersive experience.
- Accent color: Blue (#3260a1), an analogous hue to orange with slightly desaturated values, offering a cool contrast to the primary color, to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) to provide a modern and neutral user experience.
- Use a minimal, responsive, and mobile-first design approach for optimal viewing across devices.
- Employ a set of simple icons for categories and interactive elements.
- Add page transitions and card hover effects to the web application using Framer Motion for UI enhancements.
